# Data Split & Validation(MAPE)
# Split: Train(2020-2024) and Test(2025)
train_data <- window(log_retails_ts, end = c(2024, 12))
test_data  <- window(log_retails_ts, start = c(2025, 1))

# Fit model on Training set
fit_val <- auto.arima(train_data, seasonal = TRUE, 
                      stepwise = FALSE, approximation = FALSE)

# Forecast the test period (12 months)
forecast_val <- forecast(fit_val, h = 12)

# Calculate MAPE on Original Scale (Back-transform with exp)
actual_test_original <- exp(as.numeric(test_data))

The auto.arima function identified the optimal model as SARIMA(1,0,2)(0,1,1)[12] with drift. The high AR1 coefficient (0.9692) indicates a strong month-to-month dependency, while the significant seasonal moving average term (SMA1 = -0.8255) confirms the presence of consistent yearly patterns. The drift term of 0.0037 accounts for the overall growth trend in the U.S. retail sector. With a remarkably low Training MAPE of 0.14% and a highly negative AIC, the model demonstrates a superior fit, making it a reliable tool for future forecasting.

#Residual Diagnostic Analysis (checking white noise)
residuals <- residuals(fit_sarima)

# Perform the Ljung-Box test on the residuals(p-value>0.05)
ljung_box_test<- Box.test(residuals(fit_sarima), lag = 24, type = "Ljung-Box")
ljung_box_test
## 
##  Box-Ljung test
## 
## data:  residuals(fit_sarima)

this is significantly greater than the 0.05 threshold, we fail to reject the null hypothesis. This statistically proves that our residuals are independent and behave like White Noise.
